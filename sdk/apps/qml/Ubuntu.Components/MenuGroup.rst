Ubuntu.Components MenuGroup
===========================

Logical list of items for a menu.

Detailed Description
--------------------

Example usage:

.. code:: qml

    import QtQuick 2.4
    import Ubuntu.Components 1.3
    Menu {
        text: "Edit"
        MenuGroup {
            Action { text: "Undo" }
            Action { text: "Redo" }
        }
        MenuGroup {
            Action { text: "Cut" }
            ActionList {
                Action { text: "Copy" }
                Action { text: "Paste" }
            }
        }
        MenuGroup {
            Action { text: "Select All" }
        }
    }
