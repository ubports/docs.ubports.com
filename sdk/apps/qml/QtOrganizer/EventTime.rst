QtOrganizer EventTime
=====================

The EventTime element contains the start and end dates and times of a recurring event series, or occurrence of an event.

+---------------------+--------------------------+
| Import Statement:   | import QtOrganizer 5.0   |
+---------------------+--------------------------+
