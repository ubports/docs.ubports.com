Detailed Description
--------------------

The :ref:`PressureSensor <sdk_qtsensors_pressuresensor>` element reports on atmospheric pressure values.

This element wraps the QPressureSensor class. Please see the documentation for QPressureSensor for details.

**See also** :ref:`PressureReading <sdk_qtsensors_pressurereading>`.
