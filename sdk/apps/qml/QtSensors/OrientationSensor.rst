Detailed Description
--------------------

The :ref:`OrientationSensor <sdk_qtsensors_orientationsensor>` element reports device orientation.

This element wraps the QOrientationSensor class. Please see the documentation for QOrientationSensor for details.

**See also** :ref:`OrientationReading <sdk_qtsensors_orientationreading>`.
