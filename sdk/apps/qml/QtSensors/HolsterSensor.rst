Detailed Description
--------------------

The :ref:`HolsterSensor <sdk_qtsensors_holstersensor>` element reports on whether a device is holstered.

This element wraps the QHolsterSensor class. Please see the documentation for QHolsterSensor for details.

**See also** :ref:`HolsterReading <sdk_qtsensors_holsterreading>`.
