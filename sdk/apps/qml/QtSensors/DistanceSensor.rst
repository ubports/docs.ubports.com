Detailed Description
--------------------

The :ref:`DistanceSensor <sdk_qtsensors_distancesensor>` element reports the distance in cm from an object to the device.

This element wraps the QDistanceSensor class. Please see the documentation for QDistanceSensor for details.

**See also** :ref:`DistanceReading <sdk_qtsensors_distancereading>`.
