QtQuick PathElement
===================

PathElement is the base path type

+---------------------+----------------------+
| Import Statement:   | import QtQuick 2.4   |
+---------------------+----------------------+

Detailed Description
--------------------

This type is the base for all path types. It cannot be instantiated.

**See also** :ref:`Path <sdk_qtquick_path>`, :ref:`PathAttribute <sdk_qtquick_pathattribute>`, :ref:`PathPercent <sdk_qtquick_pathpercent>`, :ref:`PathLine <sdk_qtquick_pathline>`, :ref:`PathQuad <sdk_qtquick_pathquad>`, :ref:`PathCubic <sdk_qtquick_pathcubic>`, :ref:`PathArc <sdk_qtquick_patharc>`, :ref:`PathCurve <sdk_qtquick_pathcurve>`, and :ref:`PathSvg <sdk_qtquick_pathsvg>`.
