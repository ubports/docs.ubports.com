QtQuick UI Components: Dial Control Example
===========================================


This example shows how to create a dial-type control. It combines `Image </sdk/apps/qml/QtQuick/imageelements/#image>`_  elements with :ref:`Rotation <sdk_qtquick_rotation>` transforms and :ref:`SpringAnimation <sdk_qtquick_springanimation>` behaviors to produce an interactive speedometer-type dial.
